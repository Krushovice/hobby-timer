# NEURO-GUARD

Десктопное приложение для Windows. Отслеживает время в Chrome/YouTube, эскалирует напоминания о перерыве, предлагает альтернативы на основе личного профиля.

## Стадии эскалации
- **Stage 1** (30 мин) — тихое уведомление + совет
- **Stage 2** (45 мин) — оверлей с обратным отсчётом
- **Stage 3** (60 мин) — принудительное закрытие Chrome / блокировка YouTube

## Требования
- Windows 10/11
- Запуск от администратора (для hosts-блокировки)
- Gemini API ключ (бесплатный tier — [aistudio.google.com](https://aistudio.google.com))

## Разработка
```bash
npm install
npm run dev
```
